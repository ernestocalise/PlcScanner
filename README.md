# PlcScanner

PlcScanner is a Windows application for scanning, monitoring, and simulating PLC (Programmable Logic Controller) data using the OPC UA protocol. It provides a user-friendly interface to connect to OPC UA servers, visualize and log tag data, and simulate PLC environments for testing and development.

## Features

- Connect to OPC UA servers as a client
- Simulate an OPC UA server for testing
- Browse, monitor, and log PLC tag data
- Manage multiple PLC configurations
- Visualize tag values and status in real time
- Export and import configurations
- Logging and diagnostics
- On-Demand execution of advanced simulation
- Conditional Routine and Pre-programmed Routine
- Record routine directly from the PLC

## Software Installation

1. Download the installer from [here](https://www.ernestocalise.it/PlcScanner/).
2. Run the installer

## Usage

1. Launch the application.
2. Configure a new PLC connection or load an existing configuration.
3. Connect to an OPC UA server or start the built-in simulation server.
4. Browse and monitor tags, view real-time data, and log events.

## Repository Installation

1. Clone or download this repository.
2. Open `PlcScanner.slnx` in Visual Studio 2019 or later.
3. Restore NuGet packages if required.
4. Build the solution and run the `PlcScanner` project.

## Technologies Used

- C# (.NET Framework 4.7.2)
- Windows Forms
- OPC UA (OPCFoundation.NetStandard.Opc.Ua)
- Krypton Toolkit (UI components)
- Newtonsoft.Json
